# 1. Introduction

## What is FreeBSD?

FreeBSD is a UNIX-like operating system developed from the Berkeley Software Distribution (BSD). It is widely used for servers, networking, and security systems.

## Motivation

The purpose of this project is to gain practical experience installing an operating system in a virtual environment using VMware Workstation.

---

# 2. Objectives

The objectives of this project are:

- Install FreeBSD in VMware
- Understand virtualization
- Configure system settings
- Identify problems during installation
